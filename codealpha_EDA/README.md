# 📊 Netflix Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends, patterns, and insights about content distribution.

## Project Structure
- data/ → Dataset used for analysis
- notebooks/ → Jupyter Notebook containing EDA
- README.md → Project documentation
- requirements.txt → Required libraries

## Key Analysis
- Movies vs TV Shows distribution
- Content growth over years
- Top countries producing content
- Popular genres
- Missing values and data cleaning

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate over TV Shows
- USA and India are top contributors
- Significant growth after 2015
- Drama and International genres are most common
